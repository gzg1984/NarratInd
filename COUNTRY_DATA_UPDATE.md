# 国家数据更新说明

## 更新日期
2025年11月20日

## 更新内容

### 1. 补全所有国家数据
- **之前**：只有约60个主要国家的数据
- **现在**：补全了SVG地图中所有250+个国家/地区的数据
- **数据包括**：
  - 国家ID（两字母代码）
  - 人口（百万）
  - 财富等级（1-10）
  - 邻国列表

### 2. 添加邻国系统
每个国家现在都包含邻国列表，用于控制跨国传播。

**数据格式变更**：
```javascript
// 之前
['US', 330, 10]

// 现在
['US', 330, 10, ['CA', 'MX']]
```

### 3. 跨国传播机制更新
- **之前**：随机传播到任意未感染国家
- **现在**：只能传播到相邻的未感染国家
- **实现**：`gameState.js` 中的 `handleCrossBorderSpread()` 函数已更新

### 4. 新增辅助函数
在 `countryData.js` 中添加：
- `getNeighbors(country)` - 获取国家邻国列表
- `getInfectedNeighborsCount(country, countries)` - 获取已感染邻国数量

## 主要地区邻国示例

### 大国邻国配置
- **中国**：15个邻国（蒙古、俄罗斯、朝鲜、越南、老挝、缅甸、印度、不丹、尼泊尔、巴基斯坦、阿富汗、塔吉克斯坦、吉尔吉斯斯坦、哈萨克斯坦、韩国）
- **俄罗斯**：12个邻国（挪威、芬兰、爱沙尼亚、拉脱维亚、白俄罗斯、乌克兰、格鲁吉亚、阿塞拜疆、哈萨克斯坦、中国、蒙古、朝鲜）
- **巴西**：10个邻国（乌拉圭、阿根廷、巴拉圭、玻利维亚、秘鲁、哥伦比亚、委内瑞拉、圭亚那、苏里南、法属圭亚那）
- **德国**：9个邻国（丹麦、波兰、捷克、奥地利、瑞士、法国、卢森堡、比利时、荷兰）

### 岛国配置
岛国（如日本、英国、菲律宾、新西兰等）通常只有少数或没有陆地邻国：
- **日本**：['KR'] - 只与韩国有海上联系
- **英国**：['IE'] - 只与爱尔兰接壤
- **新西兰**：[] - 无陆地邻国
- **澳大利亚**：['ID', 'PG'] - 只与印尼和巴布亚新几内亚有海上联系

## 游戏影响

### 传播策略变化
1. **地理位置重要性增加**：
   - 从边缘岛国开始会很难传播到其他地区
   - 从中心大陆国家开始能更快覆盖全球

2. **建议起始国家**：
   - **中国**：15个邻国，可快速辐射亚洲
   - **俄罗斯**：连接欧亚，12个邻国
   - **德国**：欧洲中心，9个邻国
   - **巴西**：南美中心，10个邻国

3. **困难起始国家**：
   - **新西兰**：完全孤立
   - **冰岛**：无陆地邻国
   - **日本**：只能传播到韩国

### 数据验证
所有主要国家的邻国配置已验证，包括：

- 主要海上联系（如马来西亚-新加坡）

3. 某些争议地区按地理位置配置
